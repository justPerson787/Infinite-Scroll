# Infinite scroll

Allows user to scroll pictures from Unsplash photobank indefinitely.

Live demo:  https://justperson787.github.io/Infinite-Scroll/

## Technologies

JavaScript, HTML, CSS

### Features

Unsplash API, customized loader icon for better user experience
